# Zyam Maqsood — Portfolio

Personal portfolio built with Vite, React and TypeScript. This repository contains the source for my portfolio site and demo components.

## Author

Zyam Maqsood

## Quick start

Clone the repo and install dependencies, then run the dev server.

Using npm (example):

```bash
git clone <YOUR_GIT_URL>
cd Portfolio
npm install
npm run dev
# open http://localhost:8080
```

Or using pnpm:

```bash
corepack enable
corepack prepare pnpm@latest --activate
pnpm install
pnpm dev
```

Or using Bun (this repository includes a `bun.lockb`):

```bash
curl -fsSL https://bun.sh/install | bash
bun install
bun run dev
```

## Scripts

- `dev` — start Vite dev server (default port 8080)
- `build` — production build
- `preview` — preview the production build
- `lint` — run ESLint

## Notes

- The project uses a `@` alias for `./src` (configured in `vite.config.ts`).
- Tailwind CSS is configured (`tailwind.config.ts`) and utilities are included in the CSS entry files.

## Deploy

Any static-hosting provider that supports Vite-built apps works (Vercel, Netlify, GitHub Pages with proper base). Set the build command to `npm run build` and publish the `dist` folder.
